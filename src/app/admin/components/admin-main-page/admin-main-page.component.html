Witaj administratorze!

Dodawaj kolejne oferty pracy i przeprowadzaj rekrutację.

Pod zakładką 'Oferty pracy' możesz znaleźć wszystkie dodane przez Ciebie oferty.

Pod zakładką 'Nadesłane zgłoszenia' widoczne są wszystkie zgłoszenia nadesłane przez kandydatów.

Wybranie najlepszego kandydata ułatwi Ci algorytm, który nadaje odpowiednią ilość punktów każdemu z kandydatów na podstawie kilku czynników.

Punkty uzyskane przez kandydatów możesz znaleźć w kolumnie wynik.
